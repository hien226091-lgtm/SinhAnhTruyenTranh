Kịch bản

Kéo & thả file kịch bản (pdf/docx/txt/json) hoặc dán trực tiếp nội dung bên dưới.

Bộ khung (Layout JSON)

Tải lên file layout JSON hoặc kéo vùng để upload. Bạn có thể chỉnh sửa JSON trực tiếp.

Nhân vật

Kéo & thả ảnh tham chiếu cho nhân vật (hỗ trợ nhiều file). Mỗi nhân vật sẽ có mô tả và ảnh tham chiếu.

Phân tích & Xem khung truyện

Kiểm tra các khung được sinh. Bạn có thể chỉnh từng khung, sau đó xuất hoặc gửi API để render ảnh.

Phân tích hoàn tất! Nhấn "Gửi tới AI để vẽ" để bắt đầu quá trình tạo truyện.
